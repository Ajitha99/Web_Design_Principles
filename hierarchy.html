Hierarchy

As per Law of Pragnanz, we have to reduce cognitive overload so that user can use least cognitive effort possible to interpret information on the web. So avoid using complex shapes and simple structures.

As per Miller's Law, we have to align elements into relevant blocks,colums and sections, instead of throwing them all over the page. We should plan the interface of the website so that we can accomodate new features in the future while remaining easy to operate.
