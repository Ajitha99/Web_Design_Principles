Web Design Principle

We need to keep the below stated principle in our mind when designing a website. It should be user friendly, attention grabing and less complex. Let's learn more about the principles and make better websites!

Navigation Design principles

  • Simplicity : allows effective web design
  • Clear Navigation Paths : makes it easy for user to navigate between pages
  • Consistent Design : user doesn/t have to get used a new page everytime

Hierarchy Design principles

  • Contrast : Contrast helps most important parts of deisgn stand out
  • Alignment : Alignment helps create an ordered design
  • Proximity : Proximity helps user's eye follow to the next closest and dominant element

Color Design principles

  • Create Visual Balance: Color palette should create a visual harmony
  • Repetition : Repetition helps people to recognize easily - ex: colors that are on a brand
  • Consistency: Colors usage should be consistent even when the context changes
