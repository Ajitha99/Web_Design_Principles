Navigation

We should make navigation on the website easy for the user.It should not be hidden. Each webpage should tell the user in which page they are. By Hick-Hyman Law, more choices leads to more confusion to the user and will take more time to make decision. Another important point to keep in mind is, website should have only the most essential options.

The Law of common region, the elements which are grouped together closely are perceived as connected to each other by user. Example, Menu : Navigation links are formed together.
